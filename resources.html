Supported Front-end Frameworks#

  • Rax Rax is a front-end framework with React-compatible APIs.
  • Vue.js Vue.js is a progressive front-end framework for building user interfaces.

Components and Modules#

  • Weex Ui Weex Ui is a rich interaction, lightweight, high performance UI library.
  • Weex Market A website that provides a quick porting of weex’s third-party component list, where you can find the weex components you need for integration.
